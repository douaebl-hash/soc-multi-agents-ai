# Network capture (tshark - CLI Wireshark)
# Installation:
#   Debian/Ubuntu:  sudo apt-get install tshark
#   RHEL/CentOS:    sudo dnf install tshark
#   macOS:          brew install wireshark
#
# Note: tshark nécessite les permissions pour capturer sur une interface réseau.
# Solution sans sudo (après installation):
#   sudo setcap cap_net_raw,cap_net_admin=eip $(which tshark)
#   ou ajoutez l'utilisateur au groupe 'wireshark'

# Syslog socket listening: aucune dépendance, socket Python standard

# Prérequis pour le mode production:
# - tshark (network capture live)
# - iproute2 (pour détecter l'interface réseau: ip route show default)
